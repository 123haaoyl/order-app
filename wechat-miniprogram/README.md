# 微信小程序版

这个目录是原生微信小程序，不会影响根目录网页端。

## 怎么打开

1. 安装并打开微信开发者工具。
2. 选择「导入项目」。
3. 项目目录选择这个文件夹：`wechat-miniprogram`。
4. AppID 可以先选「测试号」或填你的正式小程序 AppID。
5. 导入后点「编译」即可预览。

## 目前包含

- 顾客点餐页：分类、搜索、菜品分页、规格选择、购物车。
- 美团式点餐布局：左侧分类、右侧菜品、底部购物车结算栏。
- 确认下单页：桌号 / 地址、商品明细、费用、备注、提交订单。
- 堂食 / 外卖切换。
- 桌号选择。
- 本机订单记录。
- 订单页查看历史订单。

## 接云端订单

打开 `app.js`，把：

```js
apiBaseUrl: ""
```

改成你的 Vercel 域名，例如：

```js
apiBaseUrl: "https://order-app-lemon-seven.vercel.app"
```

然后在微信公众平台后台配置：

- 「开发管理」->「开发设置」->「服务器域名」
- `request合法域名` 添加你的 Vercel 域名

配置完成后，小程序下单会请求 `https://你的域名/api/orders`，后台仍然使用现有网页后台管理订单。

## 数据来源

`utils/data.js` 是从根目录 `data.js` 生成的，当前包含 100 个菜品。以后网页端菜品如果改动，可以重新从根目录 `data.js` 同步生成这份数据。
